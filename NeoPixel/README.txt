Adafruit_NeoPixel:
The NeoPixel arduino library is used for controlling single-wire-based LED pixels and strips.


How to install: Open Arduino IDE -> Sketch -> Include Library -> Manage Libraries -> In the search box type in "NeoPixel"
-> Adafruit NeoPixel by Adafruit -> install -> restart IDE

Github: https://github.com/adafruit/Adafruit_NeoPixel
